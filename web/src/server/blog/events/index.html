A Race athlete Blog

Events are a list of community-curated endurance events. This list is all about the future. For reference purposes, past events will be featured on another page in an upcoming release.

Click on the stars next to the titles to “Favorite” events you plan to do and then filter them using the All, Star, or Owner buttons in the top right.

Below the title are the days or weeks until the event takes place, making it easy to schedule your season and training.

Don’t see an event on here you want to do? Just press the Add button in the lower right and add your own event.

If you don’t see the add button, you probably forgot to join the A Race athlete club on Strava (here).

  • Permalink to this event to share with fiends
  • Add to Google Calendar
  • Post this event on Facebook
  • Tweet about it on Twitter

When the event gets close, weather for the start location on the day of the event will appear and update as the day approaches.

Event details has all kinds of useful information such as:

  • Time Zone
  • Starting location link opening up Google Maps to help guide you
  • Strava Route Links
  • Starting Elevation
  • Link to Event homepage
