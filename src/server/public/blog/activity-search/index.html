Activity Search

What does this do?

If you have ever tried to find an activity in Strava, you have probably noticed that you may not get all of the results you expect. This search aims to solve that and deliver insights you probably didn’t even know you wanted.

By default, the page will show your activities in chronological order and you can simply keep pressing search to load more activities.

Results are displayed in three ways.

  • The status bar above the Search button will show the number of activities that are loaded on the page, the number that match the search, and the overall total.
  • The map will show pins for all of the loaded outdoor rides.
  • The list below will show the loaded rides with their stats.

Text Search

Just typing in the search box for a word or words you think you named an activity will probably work. This search uses real words, so “bike” also matches common pluralization like “bikes”. However, if you want more precision, try clicking the Precision box. Text Search also allows you to return your activities sorted by all kinds of metrics. Search defaults to Descending Date, but if you want to sort by biggest Suffer Score, just use the Sort drop down.

Location Search

Ever want to find all the rides you did while on vacation or while visiting the in-laws? Switch over to Location Search by clicking the LOCATION SEARCH tab on the top right. Click anywhere on the map to drop a pin and search from there. Press the location button just above the map to show your current location. By default it will search a 80 km/50 mi radius, but you can change this by typing in a new Max Distance. Click Search and pins will be dropped indicating the loaded results on the page. Location results are limited to the first one hundred entries.

Note: A Race athlete does not store your current location; this information is only used locally on your device. If you choose not to enable browser location, we will use the City from your Strava profile as your location.

Filters

Filters work for both Text and Location searches. Click the “filters” toggle to open the filter panel on the right. At the top you can make commutes and trainer rides Exclusive or Excluded. Default is inclusive.

Below the check boxes are the range filters. Range filters will only include results within the range you specify. You can specify dates, distance, elevation, and a host of other ranges. If you want to know all the rides between certain dates, for instance while you were riding the Tour de France last year, just set the date ranges for those dates and search will return those three weeks. Perhaps you want to know all of the rides you’ve done longer than 100 miles? Move the left distance slider to 100 and search and it will return all rides over 100 miles.

Note: Trainer rides - Currently Zwift and other Virtual Ride software do not match Trainer because Strava has reclassified them as Virtual Rides and doesn’t turn on the trainer option for the activity. Look for an update to fix this in A Race athlete in the near future.

Activity Stats

Each Activity will be displayed with metrics and link to original Strava activity. To switch your measurement preferences in real time, use the Measurement Preference Toggle in the main menu, see (this) for more information.

Download

The Download button lets you download all of the results for that search as a CSV (they do not have to be loaded on the page). If you use the download button for the initial search, it will download all of your activities.
